**Study Plan (React / React Native)**

-----
**🔹 PHASE 1 — Strong Foundations (Weeks 1–4)**

**✅ Goal**

Think like a developer, not just write code.

-----
**📅 Week 1 — JavaScript Core (MOST IMPORTANT)**

**Study (Daily)**

- Closures
- Hoisting
- Scope (block / function)
- this
- Event Loop
- Promises vs async/await

**Practice**

- Rewrite callback code into async/await
- Explain concepts **out loud** (pretend interview)

**Output**

- Notes in your own words
- 10–15 JS interview questions solved

⏱️ Time split:

- 60% understanding
- 40% coding
-----
**📅 Week 2 — Advanced JavaScript + Clean Code**

**Study**

- Array methods (map, filter, reduce)
- Immutability
- Shallow vs deep copy
- Debounce & throttle
- Error handling

**Practice**

- Build small utilities (search filter, pagination)
- Refactor old code

**Output**

- 2–3 reusable JS utilities
- Clean, readable code
-----
**📅 Week 3 — React Core (Job-Level)**

**Study**

- Component lifecycle
- useEffect (dependencies deeply)
- Controlled vs uncontrolled components
- Lifting state up
- Props drilling vs Context

**Practice**

- Build:
  - Form with validation
  - Reusable modal
  - Table with filters

**Output**

- Small React app (NO tutorial)
-----
**📅 Week 4 — React Optimization + Patterns**

**Study**

- useMemo / useCallback
- Component composition
- Folder structure
- Separation of concerns

**Practice**

- Refactor Week 3 app
- Remove unnecessary re-renders

**Output**

- Interview-ready explanations
